# Spotify-inspired Music Player

This is a simple music player that mimics Spotify's aesthetic with a few key design choices:

## Design Changes Made

1. **Color Scheme**:
   - Changed primary color to Spotify green (#1DB954) 
   - Updated background to dark theme similar to Spotify
   - Adjusted accent color for hover effects

2. **Background**:
   - Implemented dark gradient background (from #121212 to #181818)
   - Added subtle texture to the player background

3. **Styling Improvements**:
   - Rounded corners with 16px radius for modern mobile feel
   - Added subtle borders and shadows for depth
   - Enhanced button hover effects with scale animation
   - Improved spacing and layout for better readability

4. **Typography**:
   - Used clean, modern font stack
   - Improved text contrast against dark backgrounds

5. **Interactive Elements**:
   - Added subtle shadows and transitions
   - Improved progress bar aesthetics
   - Enhanced control buttons with shadow effects

## Files Modified

- `music-player/style.css`: Updated to use Spotify-inspired colors and styling
- `music-player/index.html`: Updated to match updated CSS styling

## Preview

The player now has Spotify's signature dark theme with green accents, giving it a familiar and professional look while maintaining all the original functionality.
